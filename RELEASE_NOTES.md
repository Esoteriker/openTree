# Release Notes

## 0.1.0 - 2026-02-04

### Highlights
- End-to-end scaffold for dialogue-driven knowledge graph building.
- Service boundaries and contracts documented for rapid iteration.

### Backend
- Dialogue, parser, graph, and suggestion services with FastAPI.
- Optional transformer-backed parsing contract support.
- Docker Compose stack with Neo4j, Elasticsearch, Postgres, and Redis.

### Frontend
- UI shell for session creation and graph visualization.
- Browser extension and VS Code extension starters.
