# Frontend Scaffold

## Packages

- `packages/ui-shell`: shared Web Components prototype UI.
- `packages/browser-extension`: WebExtension starter.
- `packages/vscode-extension`: VS Code extension starter.

## UI shell

```bash
cd packages/ui-shell
python3 -m http.server 4173
```

Open `http://127.0.0.1:4173`.

## Browser extension

Load `packages/browser-extension` as an unpacked extension in Chrome/Edge.

## VS Code extension

Open `packages/vscode-extension` in VS Code and run `F5` from Extension Development Host.
